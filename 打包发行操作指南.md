# 汽车仪表盘系统 — 打包发行操作指南

## 1. 发行物概述

本系统由两个独立可执行程序组成，需要分别打包。发行包为绿色免安装版本，将所有依赖 DLL 与 exe 放在同一目录，双击即可运行。

| 发行包 | 程序名 | 类型 | 端口 |
|:---|:---|:---|:---:|
| 模拟器 | `final_simulator.exe` | TCP 服务端，控制台程序 | 5000 |
| 仪表盘 | `dashboard.exe` | GUI 窗口程序，TCP 客户端 | — |

---

## 2. 环境信息

| 项 | 值 |
|:---|:---|
| Qt 版本 | 5.3.2 |
| 编译器 | MinGW 4.8.2 (32-bit) |
| Qt 安装路径 | `D:\QT\Qt5.3.2\5.3\mingw482_32\bin\` |
| MinGW 安装路径 | `D:\QT\Qt5.3.2\Tools\mingw482_32\bin\` |

---

## 3. 模拟器打包清单

### 3.1 可执行文件

从构建产物中取得，或干净构建：

```cmd
cd D:\QT\dzy\final_simulator
set PATH=D:\QT\Qt5.3.2\5.3\mingw482_32\bin;D:\QT\Qt5.3.2\Tools\mingw482_32\bin;%PATH%
set QMAKESPEC=win32-g++
qmake final_simulator.pro -spec win32-g++
mingw32-make -j4
```

产出文件：`release\final_simulator.exe`（约 40KB）

### 3.2 Qt 运行时 DLL

从 `D:\QT\Qt5.3.2\5.3\mingw482_32\bin\` 复制以下文件到 exe 同级目录：

| 文件名 | 大小 | 说明 |
|:---|:---:|:---|
| Qt5Core.dll | 4.6MB | Qt 核心库 |
| Qt5Network.dll | 1.4MB | TCP 网络通信 |
| Qt5Gui.dll | 4.5MB | Qt GUI 基础（Qt5Network 间接依赖） |
| icudt52.dll | 23.5MB | Unicode 数据表 |
| icuin52.dll | 3.3MB | Unicode 国际化 |
| icuuc52.dll | 2.0MB | Unicode 基础 |

### 3.3 MinGW 运行时 DLL

从 `D:\QT\Qt5.3.2\Tools\mingw482_32\bin\` 复制以下文件到 exe 同级目录：

| 文件名 | 大小 | 说明 |
|:---|:---:|:---|
| libgcc_s_dw2-1.dll | 115KB | GCC 运行时 |
| libstdc++-6.dll | 948KB | C++ 标准库 |
| libwinpthread-1.dll | 48KB | Windows 线程库 |

### 3.4 配置文件

在 exe 同级目录创建 `qt.conf`：

```ini
[Paths]
Prefix = D:/QT/Qt5.3.2/5.3/mingw482_32
Plugins = plugins
```

> 模拟器是控制台程序，不需要 `platforms/qwindows.dll`。qt.conf 作为保留配置留用。

### 3.5 启动脚本

在 exe 同级目录创建 `run.bat`：

```bat
@echo off
cd /d "%~dp0"
start "final_simulator" "%~dp0final_simulator.exe" %*
```

## 4. 仪表盘打包清单

### 4.1 可执行文件

干净构建：

```cmd
cd D:\QT\dzy\work\dashboard
set PATH=D:\QT\Qt5.3.2\5.3\mingw482_32\bin;D:\QT\Qt5.3.2\Tools\mingw482_32\bin;%PATH%
set QMAKESPEC=win32-g++
qmake dashboard.pro -spec win32-g++
mingw32-make -j4
```

产出文件：`release\dashboard.exe`（约 110KB）

### 4.2 Qt 运行时 DLL

从 `D:\QT\Qt5.3.2\5.3\mingw482_32\bin\` 复制以下文件到 exe 同级目录：

| 文件名 | 大小 | 说明 |
|:---|:---:|:---|
| Qt5Core.dll | 4.6MB | Qt 核心库 |
| Qt5Gui.dll | 4.5MB | Qt GUI 绘图 |
| Qt5Widgets.dll | 5.7MB | Qt 窗口控件 |
| Qt5Network.dll | 1.4MB | TCP 网络通信 |
| Qt5Multimedia.dll | 751KB | 音效播放 |
| icudt52.dll | 23.5MB | Unicode 数据表 |
| icuin52.dll | 3.3MB | Unicode 国际化 |
| icuuc52.dll | 2.0MB | Unicode 基础 |

### 4.3 MinGW 运行时 DLL

同模拟器，从 `D:\QT\Qt5.3.2\Tools\mingw482_32\bin\` 复制：

| 文件名 | 大小 |
|:---|:---:|
| libgcc_s_dw2-1.dll | 115KB |
| libstdc++-6.dll | 948KB |
| libwinpthread-1.dll | 48KB |

### 4.4 字体文件

如果不部署字体，系统会自动查找系统已有字体。如需确保 Arial 可用，可放置 Arial 字体文件，但通常不需要，Windows 自带 Arial。

### 4.5 平台插件

仪表盘是 GUI 程序，必须部署 `qwindows.dll` 平台插件。注意插件必须放在名为 `platforms` 的子目录中。

在 exe 同级创建 `platforms\` 目录，复制以下文件：

| 源路径 | 目标路径 |
|:---|:---|
| `D:\QT\Qt5.3.2\5.3\mingw482_32\plugins\platforms\qwindows.dll` | `发行目录\platforms\qwindows.dll` |

如果仪表盘菜单的 `qwindows.dll` 找不到，程序将无法打开窗口，进程会静默退出无任何提示。这是最容易被忽略的部署错误。

### 4.6 资源目录

将项目中的以下目录连同内容复制到 exe 同级：

```text
发行目录\
├── dashboard.exe
├── ... (DLL 文件)
├── images\               ← 全部复制
│   ├── background.png
│   ├── speed.png
│   ├── abs.png
│   ├── airbag.png
│   ├── battery_fault.png
│   ├── brake.png
│   ├── fuel.jpeg
│   ├── temp.jpg
│   ├── left.png
│   ├── right.png
│   └── oil.png
└── sounds\               ← 全部复制
    ├── engine.wav
    ├── oil.wav
    ├── battery.wav
    ├── temp.wav
    ├── fuel.wav
    ├── brake.wav
    ├── airbag.wav
    └── abs.wav
```

两个资源目录必须与 exe 在同一级。程序运行时通过相对路径 `images/xxx.png` 和 `sounds/xxx.wav` 加载资源。如果目录缺失或文件不全，程序仍可运行，但对应的图片和音效将为空白或静音。

### 4.7 配置文件

在 exe 同级目录创建 `qt.conf`：

```ini
[Paths]
Prefix = D:/QT/Qt5.3.2/5.3/mingw482_32
Plugins = plugins
```

> 此处的 Prefix 必须与实际 Qt 安装路径一致，使程序能找到 platforms 插件和其他资源。

### 4.8 启动脚本

在 exe 同级目录创建 `run.bat`：

```bat
@echo off
cd /d "%~dp0"
start "dashboard" "%~dp0dashboard.exe" %*
```

## 5. 一键打包脚本

以下 PowerShell 脚本可从开发目录自动打包两个程序到指定的输出目录。修改 `$devRoot` 和 `$publishRoot` 变量为实际路径后执行。

```powershell
# 配置路径
$devRoot    = "D:\QT\dzy"
$qtBin      = "D:\QT\Qt5.3.2\5.3\mingw482_32\bin"
$mingwBin   = "D:\QT\Qt5.3.2\Tools\mingw482_32\bin"
$pluginsSrc = "D:\QT\Qt5.3.2\5.3\mingw482_32\plugins\platforms\qwindows.dll"
$publishRoot = "D:\publish"

# 创建发行目录
$simDir   = Join-Path $publishRoot "final_simulator"
$dashDir  = Join-Path $publishRoot "dashboard"
New-Item -ItemType Directory -Force -Path $simDir, $dashDir

# 模拟器打包
Copy-Item "$devRoot\final_simulator\release\final_simulator.exe" $simDir
Copy-Item "$qtBin\Qt5Core.dll"     $simDir
Copy-Item "$qtBin\Qt5Network.dll"  $simDir
Copy-Item "$qtBin\Qt5Gui.dll"      $simDir
Copy-Item "$qtBin\icudt52.dll"     $simDir
Copy-Item "$qtBin\icuin52.dll"     $simDir
Copy-Item "$qtBin\icuuc52.dll"     $simDir
Copy-Item "$mingwBin\libgcc_s_dw2-1.dll"   $simDir
Copy-Item "$mingwBin\libstdc++-6.dll"      $simDir
Copy-Item "$mingwBin\libwinpthread-1.dll"  $simDir

# 模拟器 qt.conf
@"
[Paths]
Prefix = D:/QT/Qt5.3.2/5.3/mingw482_32
Plugins = plugins
"@ | Out-File -FilePath (Join-Path $simDir "qt.conf") -Encoding ASCII

# 模拟器 run.bat
@"
@echo off
cd /d "%~dp0"
start "final_simulator" "%~dp0final_simulator.exe"
"@ | Out-File -FilePath (Join-Path $simDir "run.bat") -Encoding ASCII

Write-Host "[OK] final_simulator packed to: $simDir"

# 仪表盘打包
Copy-Item "$devRoot\work\dashboard\release\dashboard.exe" $dashDir
Copy-Item "$qtBin\Qt5Core.dll"     $dashDir
Copy-Item "$qtBin\Qt5Gui.dll"      $dashDir
Copy-Item "$qtBin\Qt5Widgets.dll"  $dashDir
Copy-Item "$qtBin\Qt5Network.dll"  $dashDir
Copy-Item "$qtBin\Qt5Multimedia.dll" $dashDir
Copy-Item "$qtBin\icudt52.dll"     $dashDir
Copy-Item "$qtBin\icuin52.dll"     $dashDir
Copy-Item "$qtBin\icuuc52.dll"     $dashDir
Copy-Item "$mingwBin\libgcc_s_dw2-1.dll"   $dashDir
Copy-Item "$mingwBin\libstdc++-6.dll"      $dashDir
Copy-Item "$mingwBin\libwinpthread-1.dll"  $dashDir

# 平台插件
New-Item -ItemType Directory -Force -Path (Join-Path $dashDir "platforms")
Copy-Item $pluginsSrc (Join-Path $dashDir "platforms\qwindows.dll")

# 资源目录
Copy-Item "$devRoot\work\dashboard\images" (Join-Path $dashDir "images") -Recurse
Copy-Item "$devRoot\work\dashboard\sounds" (Join-Path $dashDir "sounds") -Recurse

# 仪表盘 qt.conf
@"
[Paths]
Prefix = D:/QT/Qt5.3.2/5.3/mingw482_32
Plugins = plugins
"@ | Out-File -FilePath (Join-Path $dashDir "qt.conf") -Encoding ASCII

# 仪表盘 run.bat
@"
@echo off
cd /d "%~dp0"
start "dashboard" "%~dp0dashboard.exe"
"@ | Out-File -FilePath (Join-Path $dashDir "run.bat") -Encoding ASCII

Write-Host "[OK] dashboard packed to: $dashDir"
Write-Host "[DONE] All packages ready at: $publishRoot"
```

## 7. 部署与运行

将发行包目录复制到目标机器后：

1. 双击 `final_simulator\run.bat` 启动模拟器。控制台显示 `listening on 0.0.0.0:5000`
2. 双击 `dashboard\run.bat` 启动仪表盘。窗口自动连接模拟器
3. 如需仪表盘独立运行：在命令行中执行 `dashboard.exe --mock`

目标机器上不需要安装 Qt 或任何运行时环境，所有依赖已包含在发行包内。
